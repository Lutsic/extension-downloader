## **CRX Extension Downloader**

**CRX Extension Downloader** is a lightweight and efficient tool designed to retrieve `.crx` files of extensions on Chrome Web Store.

### **🌟 Key Features**

* **Zero Installation:** Core functionality built entirely with HTML and JavaScript. It runs as a website or a local file in any web browser.  
* **Bypass Restrictions:** Specifically designed for school Chromebooks and restricted environments where the Chrome Web Store is blocked and software installation is disabled.  
* **Portable & Secure:** No admin rights or external dependencies are required for the core functionality.  
* **Python Utilities:** Includes local Python scripts for specialized use cases and advanced automation, if Python installation is possible.
